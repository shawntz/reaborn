# Plot the values in a color palette as a horizontal array

Port of `seaborn.palplot`. Returns a
[reaborn_plot](https://reaborn.org/reference/reaborn_plot.md).

## Usage

``` r
palplot(pal, size = 1)
```

## Arguments

- pal:

  A sequence of colors (e.g. from
  [color_palette](https://reaborn.org/reference/color_palette.md)).

- size:

  Scaling factor for the swatch size.

## Value

A `reaborn_plot`.
